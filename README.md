# Eccomerce-Website
## Description 
A simple e-commerce website for SawShop.

## Usage 
1. Clone the repository 
```
git clone https://github.com/yourusername/SawShop.git
```
2. Open 'SawShop/index.html' in your web browser.

## Requirements 
- Web browser (e.g. Chrome, Firefox)
